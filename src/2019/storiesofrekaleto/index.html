Welcome to the Rekaleto Summer Camp 2019 IF Library

Interactive Fiction Games written by and designed by children at Russian English Learning Summer Camp Rekaleto 2019 then built in Twine by Chris Godber.

Scroll through the stories using the scroller below and select play to play the games
